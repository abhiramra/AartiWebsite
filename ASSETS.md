## How to deploy
1. Copy all your images into `assets/images/`
2. Copy all PDFs into `assets/pdfs/`
3. Push the whole folder to GitHub Pages or drag-and-drop to Netlify

## Image Guidelines
- **Format:** JPG for photos, PNG for logos/icons with transparency
- **Hero images:** minimum 1400×600px (landscape); compress to ≤300KB
- **Card/news images:** 800×450px (16:9 ratio); ≤100KB
- **Profile/board photos:** 400×400px (1:1 square); ≤80KB
- **Voice portraits:** 600×450px (4:3 ratio); ≤120KB
- **Empower collage:** 400×400px (1:1 square); ≤80KB each
- **Pillar icons:** 120×120px transparent PNG; ≤30KB each
- **Logo:** ideally SVG or high-res PNG; ≤50KB
- **CTA circular image:** 400×400px minimum for crisp circle crop

## Updating Content
Each HTML file has `<!-- CONTENT: ... -->` comments marking text and values
that editors are most likely to need to update. Search for `CONTENT:` across
all files to find them quickly.

## Adding a New Voice Story Page
1. Duplicate `about-voicesofaarti-nagaveni.html`
2. Rename to `about-voicesofaarti-[firstname].html`
3. Update the name, subtitle, image src, and story text
4. Add a new `<div class="voice-card">` block in `about-voicesofaarti.html`
   pointing to the new page
